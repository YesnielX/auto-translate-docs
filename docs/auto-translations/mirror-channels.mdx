# Mirror Channels

## # Examples

The command **`/mirror-channel`** tiene varios sub-comandos.


- **`/mirror-channel list-mirror-channels`** List the mirror channels
- **`/mirror-channel add-channel <channelId> <channelId>`** Add channel to mirror channels
- **`/mirror-channel delete-channel <channelId> <lang>`** Delete a channel from mirror channels
