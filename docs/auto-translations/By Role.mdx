# By Role

This functionality allows each user of your server to configure their role to automatically translate.

##  How does it work ?

The roles configuration is done as follows.

### ** Configure your role to automatically translate. **

## # Command use

```
/mylang {Your Language}
```

![](</img/docs/by-role-setup.png>)

The Bot will create a role if it does not exist and will be assigned to the user.

When someone with another configured language sends a message, the bot translates the message to the other languages ​​of other users.
